# Example Translation Application

## Clone Project

```sh
git clone https://github.com/antheiz/example-translation-app.git
```

## How to run this Project

**With Net/HTTP**

```sh
cd terjemahan/
go run main.go
```

**With Echo**

```sh
cd echo/
go run main.go
```

**With Fiber**

```sh
cd fiber/
go run main.go
```
