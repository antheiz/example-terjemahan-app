## Clone this Project

```sh
git clone https://github.com/antheiz/example-translation-app.git
cd example-translation-app/
```

## How to run this Project

**Net/HTTP**

```sh
cd translation/
go run main.go
```

**Echo**

```sh
cd echo/
go run main.go
```

**Fiber**

```sh
cd fiber/
go run main.go
```
